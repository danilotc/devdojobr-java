Associação pt 04 - Associação bidirecional

Nesta aula foi mostrado como realizar uma relação entre classes de forma bidirecional, essa relação é chamada de associação e é bidirecional porque a relação é feita da classe A para a classe B e vice-versa.

A classe Time ganhou um novo atributo para armazenar uma lista de objetos do tipo Jogador. Com ele também foi criado um construtor e um método para mostrar os dados. Dentro desse método tem um fluxo que verifica se o novo atributo está nulo para decidir se é o momento de sair do método ou continuar executando as outras instruções. Essa verificação é necessária para evitar o erro chamado NullPointerException, pois se for nulo significa que o objeto em questão nem foi instanciado.
